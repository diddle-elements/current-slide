Current Slide component

Below is an example of the component with 5 slides in the deck. The component is useful for situations where you have a custom slide deck already implemented but require some visual feedback as to where in the deck a reader is at the current moment.

Note: this component is visual-only and does not provide handlers for navigation of your slide deck. Navigation should be implemented elsewhere in your code with this component providing a visual indicator of your navigation's current state.

Usage

<html> <head> <script src="path/to/platform.js"></script> <link rel="import" href="path/to/current-slide.html" /> </head> <body> <current-slide count="5" active="1"></current-slide> </body> </html>
count
The number of slides in your stack
active
Which numbered slide is attributed with the "active" flag for styling
